Crucigrama
Práctica de XP y Refactorización para la materia de Sistemas de Información II UMSS
Generación de Crucigrama creado a partir de palabras ingresadas.

Requisito
Lenguaje: Java 8 o superior JDK version 21.0.7
Herramienta: (IDE) Intellij IDEA
Dependencias: JUnit version 5.10.0
Configuración en IntelliJ IDEA:
1. Ir a File → Project Structure → Modules → Dependencies.
2. Hacer clic en el botón "+" → Library → From Maven...
3. Escribir en el buscador: org.junit.jupiter:junit-jupiter:5.10.0
4. Confirmar y esperar a que se descargue la librería.

Uso
Para ejecutar el programa ejecutar Crucigrama.jar

Tecnologías
Java
Swing para GUI
